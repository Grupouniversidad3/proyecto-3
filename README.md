# Directorio de Nomina (RRHH) - Equipo 3

Este es un programa de consola desarrollado en JavaScript (Node.js) para gestionar los trabajadores de una empresa y sacar las cuentas de la nomina quincenal. Permite registrar, ver el personal, hacer cambios en los datos, eliminar registros de forma segura y ver estadisticas de los costos.
